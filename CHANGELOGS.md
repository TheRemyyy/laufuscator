# Changelog

All notable changes to **Laufuscator** will be documented in this file.

## [1.0.0] - 2025-12-29

### Added
- **Initial Release**
- **VM Engine**: Custom bytecode compiler and polymorphic interpreter.
- **Control Flow**: Flattening, Opaque Predicates, and MBA transformations.
- **Security Suite**: Anti-Debug, Anti-Tamper (CRC), and Integrity checks.
- **Encryption**: Multi-layer string and bytecode encryption with dynamic keys.
- **Nested VM**: Support for multiple layers of virtualization.
- **Configuration**: Rich JSON-based configuration system.
- **Documentation**: Comprehensive guides for all features and internals.
